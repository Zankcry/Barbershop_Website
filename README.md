A modern, responsive, and SEO-optimized barbershop and salon website showcasing professional grooming services in Angeles City, Pampanga.

## Overview

Charlie's Barber & Salon features a high-performance static website highlighting services, portfolio, and an integrated blog section. It is designed to provide an excellent user experience across all devices while maintaining strong local SEO visibility.

## Key Features

- **Responsive Design:** Fully responsive layout ensuring a seamless experience on mobile, tablet, and desktop devices.
- **Service Showcase & Portfolio:** Dedicated sections for haircuts, styling, and color treatments, complete with a visual gallery.
- **Static Blog:** Integrated static blog focusing on hair care education and local SEO optimization.
- **Performance Optimized:** Preloaded assets, lazy-loaded images, and deferred scripts for fast load times.
- **SEO Optimized:** Comprehensive metadata, semantic HTML5 structure, and structured data (JSON-LD) for better search engine visibility.
- **Location & Contact Integration:** Embedded Google Maps and direct links to social media for easy appointment booking.

## Tech Stack

- **HTML5:** Semantic markup and structured data.
- **Tailwind CSS:** Utility-first CSS framework for custom styling.
- **JavaScript (Vanilla):** For interactive elements like scroll animations and deferred loading.
- **AOS (Animate On Scroll):** Lightweight animation library for scroll-triggered reveal effects.
- **Google Fonts:** Utilizing the *Outfit* typeface.
- **RemixIcon:** Clean and modern icon set.

## Project Structure

```text
├── assets/          # External libraries (AOS, RemixIcon)
```
